# Bordeaux Python Meetup ![Logo](./logo-py-bdx.png ':size=100')

À propos du langage de programmation Python : partage d'expérience, de projets, de bonnes pratiques. On peut parler de tout ce qui tourne autour de Python : développement Web, outils d'automatisation, sysadmin, data, visualisation, jeux, etc.

🌍 Site web :  https://www.meetup.com/py-bdx/

Le calendrier des évènements est disponible au format iCal.
Voici son URL : [https://www.lagrappenumerique.fr/bordeaux-python-meetup/events.ics](/bordeaux-python-meetup/events.ics ':ignore')
