Les Google Developers Groups (GDGs) sont des regroupements de personnes intéressées par les technologies et plateformes de Google: Android, Chrome, Google Apps, GWT, AppEngine... partout dans le monde.

L’objectif du GDG Bordeaux est d’organiser des évènements de manière régulière :

• Conférences sur les technologies Google, Web, Cloud, Mobile...

• Des apéros autour de sujets centraux afin de se rencontrer et boire un verre ensemble ;)

• Des concours (Hackathon...)

Le calendrier des évènements est disponible au format iCal.
Voici son URL : [https://www.lagrappenumerique.fr/gdg-bordeaux/events.ics](/gdg-bordeaux/events.ics ':ignore')
